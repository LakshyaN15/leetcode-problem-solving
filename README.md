## 🗂️ How this repo is organized

Each topic folder holds one file per problem, named `NNNN-problem-slug.py` (the LeetCode problem number, zero-padded, plus the slug). Example:

```
array/
├── 0001-two-sum.py
├── 0011-container-with-most-water.py
└── README.md
hash-table/
├── 0049-group-anagrams.py
└── README.md
```

Each solution file starts with a header block:

```python
# 1. Two Sum  |  Easy
# https://leetcode.com/problems/two-sum/
# Pattern: Hash Table
# Time: O(n)  Space: O(n)
```
